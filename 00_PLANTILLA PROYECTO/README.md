# Tuned Mass Damper (TMD) – Machine Bridge

Objetivo: diseñar, prototipar y validar un TMD para reducir vibración en el puente de la máquina.

Estructura del repo
- docs/: documentación técnica y decisiones
- data/: mediciones (raw no se toca)
- hardware/: CAD, planos, BOM
- src/ + scripts/: análisis/simulación/procesado
- results/: figuras y exportables

Cómo trabajar
- Todo trabajo va ligado a un Issue.
- Los resultados se regeneran con scripts cuando aplique.
- Cambios grandes (diseño, parámetros, criterios) se registran en docs/00_admin/decisions.

Estado

- Diseño: pendiente/en curso/completada
- Prototipo: v0/v1/v2
